## Why

Componentes atuais usam Tailwind puro, exigindo desenhar cada elemento manualmente. shadcn/ui oferece componentes prontos com visual profissional, acessibilidade automática e customização total.

## What Changes

- Instalar shadcn/ui com componentes: Card, Button, Input, Select, Dialog, Badge
- Revisar todas as páginas para usar componentes shadcn
- Manter dark mode existente

## Capabilities

### New Capabilities

- `shadcn-setup`: Instalação e configuração do shadcn/ui
- `ui-components`: Componentes Card, Button, Input, Select, Dialog, Badge

### Modified Capabilities

- None

## Impact

- **Dependencies**: shadcn/ui + Radix UI + Lucide icons
- **Files**: Todas as páginas + BottomNav
- **Breaking**: Nenhum
