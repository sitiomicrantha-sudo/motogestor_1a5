## ADDED Requirements

### Requirement: Dashboard uses shadcn components
The Dashboard SHALL use Card for metrics, Button for actions, Dialog for modals, Select for platform.

#### Scenario: Metrics display
- **WHEN** user views active shift
- **THEN** metrics are displayed in Card components

#### Scenario: Start shift modal
- **WHEN** user taps "INICIAR TURNO"
- **THEN** Dialog opens with Input fields and Button

### Requirement: Garagem uses shadcn components
The Garagem SHALL use Card, Input, Button for motorcycle config form.

### Requirement: Config uses shadcn components
The Config SHALL use Card, Input, Button for user profile form.

### Requirement: Turnos uses shadcn components
The Turnos SHALL use Card for shift history list.
