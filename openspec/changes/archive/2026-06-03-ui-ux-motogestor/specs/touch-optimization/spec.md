## ADDED Requirements

### Requirement: Touch targets grandes para uso em moto
O sistema SHALL usar tamanhos mínimos de touch targets para uso com luvas ou condições molhadas.

#### Scenario: Botões principais
- **WHEN** botões de ação principal são exibidos
- **THEN** eles SHALL ter altura mínima de 64px

#### Scenario: Botões secundários
- **WHEN** botões secundários são exibidos
- **THEN** eles SHALL ter altura mínima de 56px

#### Scenario: Inputs de formulário
- **WHEN** campos de input são exibidos
- **THEN** eles SHALL ter altura mínima de 56px

### Requirement: Fontes grandes para leitura rápida
O sistema SHALL usar fontes grandes para leitura em movimento.

#### Scenario: Fonte do relógio
- **WHEN** o tempo de turno é exibido
- **THEN** a fonte SHALL ter tamanho de 72px

#### Scenario: Fonte das métricas
- **WHEN** métricas são exibidas
- **THEN** os valores SHALL ter fonte de 28-32px

#### Scenario: Fonte de texto secundário
- **WHEN** texto secundário é exibido
- **THEN** a fonte SHALL ter tamanho mínimo de 16px

### Requirement: Ícones de navegação grandes
O sistema SHALL usar ícones de navegação maiores para fácil identificação.

#### Scenario: Tamanho dos ícones
- **WHEN** a barra de navegação é exibida
- **THEN** os ícones SHALL ter tamanho de 28px

#### Scenario: Espaçamento dos ícones
- **WHEN** a barra de navegação é exibida
- **THEN** os ícones SHALL ter espaçamento mínimo de 24px entre si
